## 박유진의 홈페이지

### team-project talat
  
> 프로젝트 제목 및 개요   
  
- 운전자(드라이버)가 카풀을 등록하고, 등록한 여정에 대해 탑승자(라이더)가 검색을 통해 매칭되는 차량 공유 플랫폼

> 개발 기간

- 2021.05.10 ~ 2021.06.04    

> 역할

- 라이더의 여정 신청   
- 드라이버와 라이더 사이의 메시지   

> 기술 스택   

- java 11
- servlet 4.0.1
- mariadb 2.7.2
- mybatis 3.5.6
- jstl 1.2
- Apache Tomcat 9.0   

> 주요 개발 내용   

- JSP와 서블릿 기술을 이용하여 라이더의 여정 검색, 상세보기, 신청, 목록보기 기능 구현
- JSP와 서블릿 기술을 이용하여 드라이버와 라이더 사이의 메시지 기능 구현
